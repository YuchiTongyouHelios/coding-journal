# Day 29: Hello, Web! (Month 2, Day 1)

## What I Built Today
My first HTML page with a touch of CSS. It has a heading, a list about me, and a paragraph – plus a light blue background and navy heading.

## What I Learned
- **HTML is the structure** – tags like `<h1>`, `<ul>`, `<li>`, `<p>` hold the content.
- **CSS is the style** – I added a `<style>` block in the `<head>` to change colors.
- **Live Server is magic** – it shows my changes instantly when I save.

## My Code
```html
<!DOCTYPE html>
<html>
    <head>
        <title>Helios - Day 1</title>
        <style>
            body { background-color: lightblue; }
            h1 { color: navy; }
        </style>
    </head>
    <body>
        <h1>Hello, I'm Helios</h1>
        <ul>
            <li>I'm learning HTML & CSS after one month of Python.</li>
            <li>I built a Number Guesser, a To-Do List, and more.</li>
            <li>I'm following the Helios Protocol – typing every line myself.</li>
        </ul>
        <p>Today is sunny, and I just wrote my first web page!</p>
    </body>
</html>
```
## Key Takeaway
I typed every single character – no copy‑paste. The browser shows exactly what I tell it. HTML is just labeled boxes; CSS paints them. I can already see how my PeerTutor app's login page is built from these same pieces.
